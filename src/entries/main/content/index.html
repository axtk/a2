Intro

This web app combines several self-contained entry points with different rendering strategies:

  • React demo
    Server-side rendering (SSR) + client-side rendering (CSR) with React
    Specific dependencies: react* from package.json

  • React demo (typed routes)
    Same as above, with a slight code change to enable type-safe routes
    Specific dependencies: react*, url-shape, zod

  • Files demo
    Rendering from static files

  • Blank demo
    Rendering content generated on the server
